#### 说下`&`和`&&`的区别？

答：

1. 首先`&`和`&&`都可以作为逻辑与的运算符；区别是`&&`具有短路功能，如果`&&`符号左边的表达式为`false`，则不再计算第二个表达式

2. `&`可以用作位运算符，当`&`两边的表达式不是`boolean`类型时，`&`表示按位与的操作

   `1 & 1 = 1`

   `1 & 0 = 0`

   `0 & 0 = 0`	

   例如：

   ```java
   public class Test {
       public static void main(String[] args) {
           // 0x7A : 0111 1010
           // 0x53 : 0101 0011
           // 0x7A & 0x53 = 0101 0010 = 0x52
           System.out.println(Integer.toHexString(0x7A & 0x53));
       }
   }
   ```

   结果输出：

   ```
   52
   ```
